## Project Description
This project is an interactive, stylized fan site inspired by the legendary status of Chuck Norris.

It combines humor, pop culture, and front-end development to create a cinematic, jungle-themed web experience. The application features dynamic content, user interaction, and animated elements to bring the “myth of Chuck” to life.

---

## Problem Domain
Modern websites often lack personality and creativity when demonstrating technical skills.

This project solves that by:
- Creating an engaging, themed experience
- Combining storytelling with interactivity
- Demonstrating core front-end skills in a fun, memorable way
- Delivering Facts about Chuck Norris

---

## Features
- Animated jungle hero banner with cartoon Chuck
- Image carousel of iconic moments
- “History of the Roundhouse” timeline
- Chuck Norris facts generator
- Embedded video player
- Interactive “Sensei Vote”
- “Seagal Threat Level” humor module
- Animated roaming Chuck sprite
- Persistent state (localStorage)
- About page

---

## Technologies Used
- HTML5
- CSS3 (animations, layout, styling)
- JavaScript (DOM manipulation, events, localStorage)
