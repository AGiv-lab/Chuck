# Chuck Norris — World Phenomenon

##  Project Description
This project is a humorous, interactive web page celebrating Chuck Norris, martial arts legends, and internet meme culture. It combines dynamic JavaScript features with a structured layout and themed styling inspired by action movies and parody.

---

##  Features

###  Facts Generator
- Displays an initial set of Chuck Norris facts
- “Summon New Fact” button adds a random fact to the table
- Users can submit their own facts
- User submissions are limited to 3 entries
- New facts are appended to the table and stored in the array

---

### ⚠️ Seagal Threat Level Selector
- Radio button selection system
- Dynamically updates response text based on selected threat level
- Adds humor through conditional messaging

---

###  Sensei Voting System
- Users select a “sensei” (Chuck, Bruce, or Seagal)
- Submit button evaluates selection
- Displays different responses depending on choice
- Prevents submission with no selection

---

###  Video Player with Switch Controls
- Embedded YouTube video player
- Two selectable clips using custom buttons:
  - “Master Bullshido”
  - “Dragon Strike”
- Buttons dynamically switch the video using JavaScript
- Active button state visually highlights the current video

---

###  Layout & Design
- Three-column responsive layout:
  - Facts panel
  - Video panel
  - Interactive right panel
- Scrollable facts table with sticky header
- Hero banner with themed background
- Image carousel section
- Styled panels with consistent spacing and design

---

###  Styling
- Custom CSS for layout, spacing, and visual hierarchy
- Consistent button styling and hover effects
- Active state styling for video controls
- Subtle theme inspired by camo / action aesthetic

---
